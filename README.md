I am the creator of the **DPEX_DJA** algorithm. It is designed to be highly effective for both meta-heuristic search and machine learning optimization.

My long-term goal is to develop a new optimization algorithm that can outperform **AdamW**. **DPEX_DJA** is currently the first version of that effort.

In graph coloring benchmarks, it has already outperformed several traditional optimization algorithms. The core idea behind **DPEX_DJA** is a hybrid approach that combines local search and global search, allowing it to balance exploration and exploitation effectively.

I have also tested the algorithm in the chemical optimization domain through the **NOVA** subnet. The results were very promising, and miners have continued using this method ever since.

Although this is only the first version, I believe the approach has significant potential for a wide range of optimization problems in both machine learning and scientific computing.
